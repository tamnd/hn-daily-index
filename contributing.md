# Contributing

This project is mostly automated, but contributions to improve the generator are welcome.

## Improving the generator

1. Fork this repo.
2. Make your changes in `src/hn_daily_index/`.
3. Run `uv run hn-daily-index` locally to verify the output.
4. Open a pull request.

## Reporting issues

If you notice missing data, formatting problems, or broken links, please [open an issue](https://github.com/tamnd/hn-daily-index/issues/new).
